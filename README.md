# Карта целей

Личный дашборд целей со стресс-тестом реалистичности.

## Локальный запуск (опционально)

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173

## Деплой на Vercel

Проект собирается через Vite. Vercel определяет это автоматически:
- Build Command: `npm run build`
- Output Directory: `dist`

Никаких переменных окружения не требуется.

## Как хранятся данные

Все данные сохраняются в localStorage браузера. Они привязаны к домену сайта и устройству. Чтобы перенести данные между устройствами — используй кнопку «экспорт» (скачивает JSON-файл) и «импорт» на другом устройстве.

Очистка кэша браузера удалит данные. Делай регулярный экспорт.
